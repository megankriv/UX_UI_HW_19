UX/UI Designer

My UX Skills

User Research

I develop empathy based research through utilizing proto-personas, empathy maps, affinity diagrams, user interviews, storyboards, journey maps, and user testing. Through thinking from a user-centered perspective I am able to solve for the root problem at the core of the user’s experience.

User Interface

I utilize design principles and theories to inform my UI design decisions. Through incorporating information architecture, sitemaps, UI systems, atomic design principles, style guides, usability testing and responsive web design, my user interfaces apply user-informed approaches to create intuitive and pleasant experiences.

Web Development

Proficient in VSCode, CodePen, and GitHub, I am capable of developing interactive and responsive web design through HTML code, CSS, and Javascript.
